Vacation Aepp

Deployed on Kovan

Vacation Aepp is a spontaneous trip-making application.

First you create a contract that holds each person funds in "escrow." This prevents people from bailing out at the last moment and making the rest of the team pay more individually

After the registration deadline or if contract fills up, an agent is able to create an itinerary for the group. The agent receives the funds sent to the contract to pay for the trip - which he uses to pay for all expenses for the trip

In the real world this application will probably needs proof of identity to assert who each user is, and also prove that each agent is qualified to do so.
